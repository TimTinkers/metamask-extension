# TextFieldSearch

The `TextFieldSearch` allows users to enter text to search. It wraps the `TextField` component that adds a search icon to the left of the input.

### Search Icon Props

`searchIconProps` allows for props to be passed to the search icon. Accepts all [Icon](/docs/ui-components-component-library-icon-icon-stories-js--default-story#props) component props.

```jsx
import { SIZES } from '../../../helpers/constants/design-system';
import { TextField } from '../../ui/component-library/text-field';

<TextFieldSearch
  searchIconProps={{ size: SIZES.MD, 'data-testid': 'search-icon' }}
/>;
```
